# neo 0.9.7 更新公告

## 重点修复

- macOS 打包流程弱化内部 Helper 的对外显示名，减少 Finder 和 Spotlight 搜索 neo 时把后台组件误认为多个 neo 应用的情况。
- 保留 Electron Helper 的内部文件名、bundle id 和后台属性，不影响应用启动、签名校验、更新流程和用户工作区数据。

## 使用提示

- Finder 或 Spotlight 里只需要关注 `/Applications/neo.app` 这个主应用。
- 如果偶尔看到 Background Service，这是 neo/Electron 的后台运行组件，不是重复安装的 neo。
