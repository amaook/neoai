# neo 0.8.9 更新公告

本次更新聚焦本地智能体执行稳定性、对话上下文复用和桌面端体验。

## 主要更新

- 工具连续调用上限从 10 轮提升到 25 轮，表格处理、文件生成、批量整理等任务更不容易在保存前中断。
- 新增对话引用：可引用单条消息或历史对话，让 neo 在当前问题中带着引用上下文继续回答。
- 新增桌宠测试能力：支持独立桌宠窗口、头像同步和桌宠设置保存。
- 新增定时任务能力：可创建定时自动化任务，按计划调用模型并输出结果。
- 增强桌面测试模式：测试版使用独立工作区和用户数据，不影响正式版数据。
- 增强环境检测与一键补齐：Windows/macOS 会检测常用本地工具，并给出补齐入口。
- 修复 Electron 本地依赖缺失导致桌面端打不开的问题，启动前会尝试自动修复 Electron 运行文件。

## 用户怎么更新

- Windows 用户：在 neo 设置中点击“检查更新”，下载完成后点击“重启安装”。
- macOS 用户：如果当前版本没有正式开发者签名，neo 会提示打开下载页手动覆盖安装。

## 说明

Windows 离线依赖包内置能力还需要补齐真实安装包资源，本版先保留联网补齐与检测能力。
