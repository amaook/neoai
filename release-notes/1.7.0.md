# neo 1.7.0 更新公告

开启新主题 **「数据表格智能」**：在 1.6「问我的文件」把知识库做厚之后，这一版让 neo 真正"看得懂表格的数据质量"。第一批落地**数据质量体检**——拿到一份脏表，先一键体检、再决定怎么清洗，是「办公最强」在数据方向的关键基建。

## 数据质量体检（check_table）

- 新增 `check_table` 工具：对 `.xlsx/.xlsm/.csv/.tsv` 表格做逐列体检，输出结构化报告 + 可直接保存的 Markdown 摘要。
- **逐列指标**：
  - 缺失率（填充/缺失数）
  - 主类型与**类型一致率**（如"金额"列里混入文本）
  - 数值列：最小/最大/均值/中位数/标准差 + **异常值（IQR 法）**
  - 日期列：**多种日期格式混用**检测
  - 文本/布尔列：Top 高频取值
  - 单元格**首尾空格**问题
- **整表指标**：完全重复行、全空行、全空列。
- **问题清单**：按严重度排序的人话提示，例如「列『金额』缺失 12%」「列『日期』存在 3 种格式，建议统一」「发现 5 行完全重复」「列『年龄』2 个疑似异常值：999、-1」。
- 数字识别更聪明：能解析千分位 `1,200`、货币符 `¥3,400`、百分号 `10%`、全角数字。

## 表格出图（chart_from_table）

- 新增 `chart_from_table` 工具：从 `.xlsx/.xlsm/.csv/.tsv` 直接出图——**折线图 / 柱状图（多系列分组）/ 饼图**。
- 选标签列（x 轴）+ 一到多个数值列（系列），不指定时**自动识别数值列**；数字解析复用体检逻辑（千分位、货币符、百分号、全角）。
- **默认输出 SVG**（带文字标签、可缩放、浏览器/Office 都能打开、不打包也能验证）；需要位图时指定 `.png/.jpg`——桌面端用排版引擎渲染、文字完整，非桌面用 canvas 兜底栅格化。
- 生成的图片进「成果区」，可打开/定位。典型场景："把月度销量画成柱状图""收入趋势图""各渠道占比饼图"。

## 用法与协同

- 接进「表格处理」「财务表格」技能：遇到可能脏的表格**先体检再清洗**，据体检结果决定删空行/去重/统一格式/核对金额；需要可视化时**直接出图**。
- 体检返回的 `markdown` 可直接交给 `write_file` 存成 `.md`、或 `create_pdf_file` 出 PDF 报告。
- 典型场景："这份工资表有没有问题？""清洗前先看看这份 CSV 的数据质量""帮我找出账单里的异常金额""把这张表画成柱状图/趋势图/占比饼图"。

## 实现与稳定性

- **纯 JS、不依赖 Python、不联网、不需要额外模型**——区别于走 openpyxl 的 `clean_table_file`，体检零环境依赖、隐私不出本机。
- 复用已有的 Excel/CSV 读取器与质量函数（`readExcelFile`/`parseDelimitedRows`/`tableQuality`），只读不写盘（仅 `fileRead` 权限，默认已开）。
- 纯新增工具，不改动现有对话主循环；新增权限位无需引入。
